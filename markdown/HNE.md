1.  **Download Data:**

      * Go to the [HNE GitHub Data folder](https://github.com/yangji9181/HNE/tree/master/Data).
      * Download the **DBLP** folder (contains `node.dat`, `link.dat`, `label.dat`, etc.).
      * Place it in your project at: `./data/HNE_DBLP/`.

2.  **Download AnyBURL:**

      * Go to the [AnyBURL website](https://web.informatik.uni-mannheim.de/AnyBURL/).
      * Download the JAR file (e.g., `AnyBURL-23-1.jar`).
      * Place it in a new folder: `./tools/AnyBURL-23-1.jar`.
      * **Requirement:** Ensure you have Java installed (`java -version`).
